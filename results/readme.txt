Папка "results" содержит результаты экспериментов по обработке данных, обучению и оценке моделей распознавания эмоций на различных комбинациях датасетов и эмоций.

Структура папок внутри "results" следует формату "датасеты_для_обучения___датасеты_для_оценки___эмоции", где:

"датасеты_для_обучения" - набор датасетов, используемых для обучения модели
"датасеты_для_оценки" - набор датасетов для оценки обученной модели
"эмоции" - список эмоций, используемых при обучении и оценке модели

Например:
Папка "CREMA-D___RAVDESS___Anger+Disgust+Fear+Happy+Neutral+Sad" содержит результаты обучения модели на датасете CREMA-D и оценки на датасете RAVDESS по тем же эмоциям.
Папка "CREMA-D+RAVDESS___TESS___Anger+Disgust+Fear+Happy+Neutral+Sad" содержит результаты обучения модели на объединенных данных из датасетов CREMA-D и RAVDESS, с последующей оценкой на датасете TESS по указанным эмоциям.
Каждая папка содержит отчеты и все необходимые файлы для воспроизведения экспериментов, включая обработку данных, обучение моделей и их оценку на соответствующих датасетах и эмоциях.
